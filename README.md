# novedraw

一個以靜態前端實作的抽獎頁面，用於「諾芙新人畢業紀念周邊抽獎」活動。

## 功能

- 輸入名稱後可進行單抽或十連抽
- 大獎結果會加入榜單
- 抽到「再來一次」會累加右上角計數器
- 榜單與計數器會透過 `localStorage` 保存在瀏覽器本機
- 支援桌面與手機版版面
- 顯示目前獎項機率

## 專案結構

```text
novedraw/
├── index.html
├── novebg.jpg
├── nove88.png
├── novehaha.png
├── novenice.png
└── assets/
    ├── css/
    │   └── styles.css
    └── js/
        └── app.js
```

## 使用方式

1. 下載或 clone 此專案
2. 直接用瀏覽器開啟 `index.html`
3. 輸入名稱後即可抽獎

這個專案目前不需要安裝套件，也不需要 build。

## 目前獎項機率

- 大獎 - 立牌（雙馬尾）：1%
- 大獎 - 立牌（長髮）：1%
- 再來一次：10%
- 哈哈 沒抽到：88%
